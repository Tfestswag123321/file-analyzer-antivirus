# file-analyzer-antivirus
Приложение на Python с графическим интерфейсом для анализа подозрительных файлов (имитация антивируса)
# Программа анализа подозрительных файлов

## Описание проекта
Данный проект представляет собой приложение на Python с графическим интерфейсом, предназначенное для первичного анализа подозрительных файлов. Программа имитирует работу простого антивирусного анализатора и помогает пользователю определить уровень риска файла по его названию и расширению.

## Какую проблему решает проект
При работе с файлами пользователь не всегда может сразу понять, безопасно ли открывать тот или иной файл. Некоторые файлы могут иметь подозрительные расширения, опасные слова в названии или признаки маскировки. Программа помогает выполнить первичную оценку файла и предупредить пользователя о возможной опасности.

## Целевая аудитория
- студенты
- начинающие пользователи ПК
- пользователи, интересующиеся основами информационной безопасности

## Цель проекта
Создать приложение с графическим интерфейсом, которое анализирует имя файла, определяет уровень риска и сохраняет результаты проверки в текстовый файл.

## Задачи проекта
- разработать удобный графический интерфейс
- реализовать ввод имени файла
- выполнить анализ названия и расширения файла
- определить уровень риска
- вывести результат пользователю
- сохранить историю проверок в файл
- реализовать несколько окон программы

## Функционал программы
- ввод имени файла или пути к файлу
- проверка расширения файла
- поиск подозрительных слов в названии
- проверка на двойное расширение
- подсчёт баллов риска
- вывод результата анализа
- очистка формы
- просмотр истории проверок
- окно «О программе»
- сохранение результатов в `log.txt`

## Используемые технологии
- Python
- Tkinter
- модуль `datetime`
- модуль `os`
